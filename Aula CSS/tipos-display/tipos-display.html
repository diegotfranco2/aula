Tipos de Display CSS

Display: Block

Block 1
Block 2
Block 3

Os elementos com display: block ocupam toda a largura disponível e são empilhados verticalmente, um abaixo do outro. Exemplos comuns são as tags <div> e <p>. Esse comportamento os torna adequados para layouts que precisam se expandir na largura.

Display: Inline

Inline 1 Inline 2 Inline 3 (sem quebra)

Os elementos com display: inline permanecem na mesma linha e não ocupam a largura inteira. Não respeitam larguras ou alturas definidas, e são comumente usados em tags como <span> e <a>. O conteúdo deles nunca quebra automaticamente para uma nova linha, a menos que o container seja pequeno o suficiente.

Display: Inline-Block

Inline-Block 1
Inline-Block 2 (pode quebrar)
Inline-Block 3

Os elementos com display: inline-block podem definir largura e altura, mas permanecem lado a lado como elementos inline. Eles podem quebrar para uma nova linha se o espaço horizontal não for suficiente. Isso os torna úteis para layouts que requerem mais controle de dimensão do que os elementos inline, mas sem a necessidade de ocupar toda a largura como block.

Display: Flex

Flex 1
Flex 2
Flex 3

O display: flex cria um container flexível, onde os itens podem ser organizados de várias formas ao longo de um eixo. Por padrão, os itens são organizados horizontalmente, mas você pode ajustar o comportamento para distribuir e alinhar os itens de acordo com as necessidades do layout. Flex é uma ótima opção para layouts responsivos.

Display: Grid

O display: grid organiza os itens em linhas e colunas, permitindo layouts mais complexos que o flex. Você pode definir o número de colunas e linhas e controlar como os itens se distribuem nessas áreas. O grid é perfeito para criar layouts bidimensionais precisos, onde você precisa controlar tanto o eixo horizontal quanto o vertical.

Display: None (você não deve ver isto)

Elemento Oculto

O display: none remove completamente o elemento do layout. O elemento não será renderizado na página e não ocupará nenhum espaço.
